WEBSITE CREATION (Beginner)

Assignment: Create a four page website. This must include a 9 image gallery, a contact form, flip cards, 3 different views, and a parallax image.

You can view the final website here

Main takeaways:

  1. Wireframing
  2. Utilization of grid and flex in CSS
  3. How to create a parallaxed imaged
  4. Navigation bar organization
  5. Media queries
  6. Accessibiliy tools (ex. skip to main content, tabbing through pages, and contrast)
  7. Mobile first design
